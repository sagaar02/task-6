# Task 6 - Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume using SQL aggregate functions.

## Tools Used
- SQLite
- SQL

## Steps Performed
1. Created the online_sales table.
2. Inserted sample sales data.
3. Calculated monthly revenue using SUM().
4. Calculated order volume using COUNT(DISTINCT order_id).
5. Grouped data by year and month.
6. Sorted results using ORDER BY.
7. Identified top 3 months by sales.

## Key SQL Concepts
- GROUP BY
- SUM()
- COUNT(DISTINCT)
- ORDER BY
- Aggregate Functions

## Outcome
Successfully analyzed sales trends and monthly performance using SQL aggregations.
